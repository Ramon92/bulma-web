### # Read only

`readonly` attribute will be delegated to the native `<input>` to make it read-only. The input will look similar to a normal one, but is not editable and has no shadow.

This field **will still be included** in the parent fieldset or form's `serializedValue`.


<Story name="Read only">
  {html`
    <bw-input readonly .modelValue=${'This is read only'} label="Read only"></bw-input>
  `}
</Story>

```html
<bw-input readonly .modelValue=${'This is read only'} label="Read only"></bw-input>
```
